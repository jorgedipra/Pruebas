```bash
npx create-nuxt-app faceLanding
```
Or starting with npm v6.1 you can do:
```bash
npm init nuxt-app@latest faceLanding
```
## To get started:

* **cd faceLanding**
* **npm run dev**

* **cd faceLanding**
